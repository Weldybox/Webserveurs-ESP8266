Graphique de température:

----------------------------ESP8266 & BME280----------------------------


Le code arduino permet de récupérer stocker et héberger le serveur web (interface utilisateur).

Le code HTML/CSS/JS récupère les données dans le fichier temperature.csv pour les afficher sous forme de graphique.
	- Le framework Highstocks permet d'afficher de manière esthétique les températures.
